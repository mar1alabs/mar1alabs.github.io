## Proyecto Final de Máster (TFM)

### RIESGOS OCULTOS: Análisis de Vulnerabilidades y Compromiso Total del Sistema

Análisis de seguridad realizado sobre la máquina vulnerable **DC:9 (VulnHub)**, simulando un entorno corporativo real.

#### Técnicas utilizadas:
- Reconocimiento y enumeración de servicios
- Explotación de vulnerabilidades web (SQLi, LFI)
- Abuso de configuración insegura (port knocking)
- Ataques de credenciales débiles
- Escalada de privilegios en Linux

#### Resultado:
Compromiso completo del sistema con obtención de acceso root.
